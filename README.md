# 🎬 Movies App

A modern and responsive movie website built using **HTML5**, **Bootstrap 5**, and **Font Awesome**.
The project showcases trending movies, upcoming releases, and a newsletter subscription section.

---

## 🚀 Features

* 🔥 Responsive Navbar with Bootstrap
* 🎥 Hero Section with background image
* 🆕 Opening This Week movies section
* ⏳ Coming Soon movies section
* 📧 Newsletter subscription form
* 📱 Fully responsive design (Mobile, Tablet, Desktop)
* 🎨 Modern dark theme UI
* 🌐 Social media footer section

---

## 🛠️ Built With

* HTML5
* CSS3
* Bootstrap 5
* Font Awesome
* Google Fonts
